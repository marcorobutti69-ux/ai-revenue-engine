# AI Revenue Engine

Piattaforma AI per Revenue Management Hotel.

## Funzioni principali

- Analisi Occupancy
- Analisi ADR
- Calcolo RevPAR
- Forecast domanda
- Suggerimento prezzo
- Simulatore ricavi

## Avvio della demo

cd ~/ai-revenue-demo

source venv/bin/activate

streamlit run app.py

Aprire il browser:

http://localhost:8501

Caricare il file:

hotel_data.csv
